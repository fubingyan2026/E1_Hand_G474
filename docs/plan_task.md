用户输入：
- 上位机/主控输入接口为 CAN 或 CAN FD
- 灵巧手控制板接收 CAN/CAN FD 指令后，转换为 2 路 UART 控制电机

电机串口拓扑：
- 使用 2 路串口控制 9 个电机
- 分组方式：5 + 4
- 每路电机 ID 建议都规划在 ID1~ID8 范围内，便于使用特殊广播帧

电机串口波特率：
- 500000 bps

电机控制：
- 使用位置 + 速度广播
- 控制频率目标 300 Hz
- 每个控制周期每路串口发送 2 帧：位置广播 1 帧 + 速度广播 1 帧

电机反馈：
- 分时轮询
- 目标每电机 100 Hz
- 优先读取 CONFIG_INFO_02_R：角度、速度、D/Q 电流
- 低频读取 CONFIG_INFO_01_R：状态、故障、温度、电压

电流限制：
- 低频配置
- 仅在动作阶段切换或限流参数变化时配置，不进入高频控制周期
